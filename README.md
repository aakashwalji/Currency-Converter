# Currency Converter

## Overview

This Currency Converter project provides a simple and user-friendly interface for converting currencies. Developed using only HTML, CSS, and JavaScript, it offers a lightweight solution without the need for additional frameworks or libraries.

## Features

- *User-Friendly Interface*: Intuitive design makes currency conversion quick and easy.
- *Real-Time Exchange Rates*: Utilizes live exchange rates to provide accurate and up-to-date conversions.

## Usage

1. Open the index.html file in your preferred web browser.
2. Select the source currency from the dropdown list.
3. Enter the amount you want to convert.
4. Choose the target currency from the second dropdown.
5. Click the "Convert" button to see the converted amount.

## Technologies Used

- *HTML*: Provides the structure and markup for the project.
- *CSS*: Handles the styling and layout of the application.
- *JavaScript*: Implements the currency conversion logic and interacts with the API for live exchange rates.

## API Integration

This project uses a third-party API to fetch real-time exchange rates. To maintain simplicity, the API key is not required, making it easy for users to get started without any additional setup.
